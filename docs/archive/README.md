# Archive

This directory holds historical material that is useful for implementation history but should not be treated as the current source of truth.

## Contents

- `analyses/`: gap analyses and one-off repo assessments
- `benchmarks/`: older benchmark notes and superseded benchmark docs
- `plans/`: implementation plans and completion notes
- `research/`: research writeups and exploratory notes
- `roadmap/`: historical roadmap material
- `verification/`: implementation verification reports

Use the current docs in [`../README.md`](../README.md) and the runnable benchmark docs in [`../../experiments/README.md`](../../experiments/README.md) for day-to-day work.
